## En esta clase se encontrará los archivos necesarios para poder crear una base de datos estándar, 
## con la base esperada para su construcción, esto no limita la creación de nuevos esquemas y tablas, la cual consiste en lo siguiente:

## 1. Esquemas bajo los cuales se encontrará segmentada la base de datos, a nivel base: Seguridad, Documento y Bitácora
## 2. El esquema Seguridad, contiene las tablas a ser utilizadas para el manejo de la seguridad de una aplicación estándar, tanto web como móvil o de escritorio.
## 3. El esquema Documento, contiene las tablas a ser utilizadas para repositorios de documentos a ser utilizados para subida de archivos en una aplicación estándar, tanto web como móvil o de escritorio.
## 4. El esquema Bitácora, contiene las tablas a ser utilizadas para el registro de movimientos, sesiones, bitácoras en otros, consideradas como estándar para cualquier tipo de aplicación.
